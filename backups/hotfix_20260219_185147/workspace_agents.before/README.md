# AGENTS - Mesa Pastoral Digital

Este diretório contém os 4 pastores especializados da Mesa Pastoral Digital, coordenados pela Pastora Chefe.

## 📁 ESTRUTURA

Cada pastor tem sua própria pasta com dois arquivos canônicos:

```
agents/
├── pastor-profetico/
│   ├── SOUL.md          # Quem ele é, voz, estilo
│   └── INSTRUCTIONS.md  # Como ele trabalha
├── pastor-consolador/
│   ├── SOUL.md          # Quem ela é, voz, estilo
│   └── INSTRUCTIONS.md  # Como ela trabalha
├── pastor-revisor/
│   ├── SOUL.md          # Quem ele é, foco
│   └── INSTRUCTIONS.md  # Como ele trabalha
└── pastor-formatador/
    ├── SOUL.md          # Quem ele é, foco
    └── INSTRUCTIONS.md  # Como ele trabalha
```

## 🔥 PASTOR PROFÉTICO

### Localização
`pastor-profetico/`

### Função
Escrever devocionais com tom firme, profético e direto

### Quando usar
- Quando o dia exige desafio, decisão ou chamado
- Quando a passagem fala de propósito, reino, obediência

### Tom
Firme, direto, profético, confronto amoroso

---

## 🤗 PASTOR CONSOLADOR

### Localização
`pastor-consolador/`

### Função
Escrever devocionais com tom acolhedor e profundo

### Quando usar
- Quando o dia exige acolhimento, cura, descanso
- Quando a passagem fala de graça, perdão, consolo

### Tom
Acolhedor, profundo, suave, bálsamo

---

## 📚 PASTOR REVISOR

### Localização
`pastor-revisor/`

### Função
Revisar validade teológica e ortográfica

### Quando usar
- Após qualquer pastor escrever um devocional
- Antes de formatar para postagem

### Foco
Precisão bíblica, correção gramatical, limpeza de rótulos

---

## 📱 PASTOR FORMATADOR

### Localização
`pastor-formatador/`

### Função
Preparar texto para postagem no WhatsApp e Instagram

### Quando usar
- Após Pastor Revisor corrigir o devocional
- Quando Lucifran pede texto pronto para postar

### Foco
Clareza visual, facilidade de leitura, layout estratégico

---

## 🔄 FLUXO DE TRABALHO

```
1. Pastora Chefe decide voz (Profético ou Consolador)
   ↓
2. Pastor escolhido escreve → Devocional bruto
   ↓
3. Pastor Revisor valida → Devocional limpo
   ↓
4. Pastor Formatador prepara → Texto pronto
   ↓
5. Pastora Chefe valida → Entrega ao Lucifran
```

## 📋 PADRÃO DE ARQUIVOS

Cada pastor deve ter:

### SOUL.md
- Nome e função
- Alma (SOUL) - filosofia e identidade
- Características
- O que faz
- O que não faz
- Firma (firma característica do pastor)

### INSTRUCTIONS.md
- Missão
- Regras de ouro (soberanas)
- Estrutura recomendada
- Checklist final
- Exemplo de tom/DNA
