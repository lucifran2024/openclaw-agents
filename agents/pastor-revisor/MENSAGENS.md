# MENSAGENS.md - Templates de Mensagens ao Usuário

Este arquivo contém templates de mensagens padrão para comunicação com o Lucifran.

---

## 💬 TEMPLATES DE RESPOSTA

### ✅ DEVOCIONAL PRONTO

**Quando devocional está pronto:**

```markdown
# Devocional - [DATA] [EMOJI]

[TEMA/LIVRO]

> ***"[VERÍCULO]"*** ([REFERÊNCIA])

[CONTEÚDO COM NEGRITOS E QUEBRAS DE LINHA]

[FRASE FINAL] [EMOJI]

---

✅ **Pronto para copiar e postar!**
```

**Interações (Botões):**
- [[text: "🔄 Refazer", callback_data: "refazer mensagem"]]
- [[text: "⭐ Favoritar", callback_data: "favoritar mensagem"]]
- [[text: "🤗 Mais Consolo", callback_data: "mais consolo"]]

### 🙏 ORAÇÃO PRONTA

**Quando oração está pronta:**

```markdown
# Oração - [DATA] 🙏

[TEMA/LIVRO]

> ***"[VERÍCULO]"*** ([REFERÊNCIA])

[ORACAO EM PRIMEIRA PESSOA]

Amém. 🙏

---

✅ **Pronto para copiar e postar!**
```

---

### 📋 AGENDAMENTO AUTOMÁTICO

**Quando sistema envia pedido automático (05:00):**

```markdown
📅 [AGENDAMENTO AUTOMÁTICO] 

São 05:00. Por favor, gere os 3 devocionais/orações para hoje seguindo:

- PLANO_LEITURA_BIBLICA.md (passagens do dia)
- PROGRAMACAO_SEMANAL.md (tipo de conteúdo)
- MEMORY.md (DNA e favoritos)

[DATA]: [DIA DA SEMANA]

Obrigado! 🙏
```

---

### 🔄 HEARTBEAT DO SISTEMA

**Quando sistema envia heartbeat (toda hora):**

```markdown
HEARTBEAT_OK

[RESUMO DO SISTEMA - se houver algo a reportar]

STATUS: Sistema operacional.
```

### 📝 FEEDBACK RECEBIDO

**Quando Lucifran envia feedback:**

```markdown

**O que você disse:**
[FEEDBACK DO LUCIFRAN]

**Ações tomadas:**
- [AÇÃO 1]
- [AÇÃO 2]

**Próximo passo:**
[O QUE VAI SER FEITO AGORA]

Obrigado! 🙏
```

## 🎯 GUIA DE USO

### Quando usar este arquivo:

1. **Resposta padrão:** Copie o template e personalize
2. **Consistência:** Mantém tom e formato consistentes
3. **Profissionalismo:** Mensagens bem estruturadas
4. **Clareza:** Informações fáceis de entender

### Como personalizar:

- Substitua `[VARIÁVEL]` pelo conteúdo real
- Ajuste o emoji conforme o contexto
- Adicione/remova informações conforme necessário
