# FORMATOS_FAVORITOS_REFERENCIA

Fonte: `FAVORITOS_COMPLETO.md` (base integral enviada por Lucifran)

## Formatos detectados para polimento
1. **Narrativa de processo (médio/longo)**
   - História bíblica + ponte para hoje + contraste forte.
2. **Confronto direto em CAIXA ALTA**
   - Frase de choque + explicação curta + versículo.
3. **Declaração/Profecia semanal**
   - "Eu profetizo..." com decretações objetivas.
4. **Oração da noite/manhã/semana**
   - Gratidão + entrega + pedido + "Amém".
5. **Devocional com título forte**
   - Título-impacto + corpo em blocos + aplicação final.
6. **Frase curta de impacto (micro-peça)**
   - 1–3 linhas, alta memorização.
7. **Versículo + aplicação curta**
   - Referência explícita e leitura prática.
8. **Lista de gratidão/repetição anafórica**
   - Estrutura repetida (ex.: "Deus, obrigado...").
9. **Alerta de discernimento**
   - "Nem tudo é batalha espiritual..." / "Discernimento...".
10. **Fechamento de ciclo / início de semana**
   - Encerrar + avançar + coragem + Cristo.

## Padrões obrigatórios de uso no polimento
- Comparar cada rodada com esta referência + `MEMORY.md`.
- Variar posição do versículo (início, meio, fim) conforme peça.
- Preservar DNA: contraste, processo, secreto/público, Cristo no centro.
- Registrar no relatório da rodada: formato aplicado e efeito percebido.
