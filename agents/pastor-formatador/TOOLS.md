# TOOLS.md - Ferramentas do Pastor Formatador

## FERRAMENTAS DISPONÍVEIS

### 📖 Leitura
- **read:** Consultar `MENSAGENS.md` para templates de formatação
- **read:** Consultar `ESTILO_DEVOCIONAL.md` para referência visual

### ✍️ Edição
- **edit:** Aplicar formatação ao texto revisado
- **write:** Salvar versão formatada

### 💬 Comunicação
- **sessions_send:** Devolver texto formatado à Pastora Chefe
- **sessions_list:** Verificar sessões ativas

---

## FLUXO DE TRABALHO

1. **Receber texto revisado** do Pastor Revisor
2. **Adicionar cabeçalho:** `📖 Leitura do dia: [PASSAGEM]`
3. **Destacar versículo:** `> ***"Texto"*** (Referência)`
4. **Aplicar negritos** nas frases mais fortes (máximo 2-3)
5. **Quebras de linha** generosas para respiração visual
6. **Emoji final** (🔥 profético, 🕊️ consolador, 🙏 oração)
7. **Enviar** à Pastora Chefe via `sessions_send`

---

## REGRAS DE FORMATAÇÃO

### WhatsApp
- Negrito: `*texto*`
- Itálico: `_texto_`
- Versículo: `> ***"Texto"***`

### Instagram Status
- Texto curto e impactante
- Sem formatação markdown
- Versículo isolado entre aspas

---

## NÃO FAZER
- Não mudar o conteúdo do texto
- Não adicionar conteúdo novo
- Não abusar de emojis (máximo 1-2)
- Não usar cores ou fontes especiais
